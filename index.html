22. m sorted increasing

`java
public List sort(int[] nums) {
List res = new LinkedList<>();
//same all the pivot
List index = new LinkedList<>();
index.add(0);
for(int i = 1; i < nums.length; i++) {
if(nums[i] <= nums[i - 1]) {
index.add(i);
}
}
PriorityQueue queue = new PriorityQueue<>((a, b)->(nums[a] - nums[b]));
Set set = new HashSet<>();

for(int i: index) {
queue.offer(i);
set.add(i);
}
set.add(nums.length);
while(!queue.isEmpty()) {
int cur = queue.poll();
//if cur + 1 is in the set, that means
//the numbers of cur sublist has been used up
if(!set.contains(cur + 1)) queue.offer(cur + 1);
res.add(nums[cur]);
}
return res;
}

6. Compressed Dot Product

给定两个压缩表示的vector 比如[(3,2),(4,1),(2,3)]
就是[2, 2, 2, 1, 1, 1, 1, 3, 3]的缩写 现在
给你两个这样压缩好的vetcor 让你求内积 当两个长度不不同的时候
按最⼩小⼀一个的长度算

`python

def vectorDotProduct(s,t):

offsetS = 0
offsetT = 0
countS = 0
countT = 0

res = 0

while True:
if countS == len(s) or countT = len(t):
break
res += s[countS][1] * t[countT][1]
offsetS += 1
offsetT += 1
if offsetS == s[countS][0]:
offsetS = 0
countS += 1
if offsetT == t[countT][0]:
offsetT = 0
countT += 1

return res
